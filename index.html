프로젝트 개요

AI 기반 인지 왜곡 분석 및 메타인지 개입 시스템 설계

학습 불안 장면에서 나타나는 자동 사고를 규칙 기반 분류와 생성형 AI 보조 해석으로 분석하고, 대안 사고와 사전·사후 비교 지표를 함께 제공하는 CBT 기반 연구 프로토타입입니다.

Hybrid 분석 규칙 기반 분류 + 생성형 AI 보조 해석
주요 왜곡 5~7종 흑백논리·과잉일반화·파국화 중심 분류
사전·사후 검증 불안 점수, 인식률, 지속률 변화를 추적
익명 로컬 저장 개인 식별 정보 없이 최소한의 데이터만 기록
현재 로컬 규칙 분석 모드입니다.
연구 동기와 목적

학습 불안 상황에서 자동 사고가 감정과 행동을 바꾸는 과정을 검증합니다

자율신경계와 스트레스 반응 탐구에서 출발해, 실제 학교생활에서 반복되는 자동 사고가 감정과 행동을 어떻게 바꾸는지 AI 기반 CBT 구조로 분석하는 연구입니다.

연구 동기

탐구 배경

생명과학 심화탐구에서 자율신경계 조절, 스트레스 생체 반응, 양측성 시각자극과 정서 상태의 관계를 살피며 정서 반응이 주의와 인지 과정과 긴밀히 연결되어 있음을 확인했습니다.

학교 장면 관찰

시험 직전 과도한 긴장, 특정 과목 실패 후 “나는 원래 못해”, 결과 직후 “망했다, 끝이다” 같은 자동 사고가 학습 회피와 낮은 자기평가로 이어지는 장면을 반복적으로 보았습니다.

핵심 연구 질문

“AI 기반 자동사고 분류 및 피드백 시스템이 학습 불안을 겪는 학생의 자기이해(메타인지), 인지적 객관화, 자기조절 효능감에 유의미한 변화를 주는가?”
사전·사후 불안 점수 사용 전후 정서 강도 변화를 비교합니다.
인지 왜곡 인식 및 수정 비율 왜곡 포착과 대안 사고 재구성 비율을 확인합니다.
기록 지속률 반복 사용을 통한 자기조절 습관 형성을 봅니다.
목표 1. 인지적 객관화

자동 사고를 사건과 분리해 스스로 점검하게 합니다.

목표 2. Hybrid 피드백

규칙 기반 분류와 생성형 AI 해석을 함께 사용합니다.

목표 3. 사전·사후 검증

불안 점수와 기록 지속률 변화를 연구 지표로 분석합니다.

프로젝트 계획

핵심 MVP 구현과 사전·사후 비교 설계를 함께 진행합니다

초기 단계에서는 기록-분류-대안 제시의 핵심 구조에 집중하고, 이후 동일 대상자의 사전·사후 비교로 시스템 효과를 검증합니다.

상황·생각·감정·행동 기록

감정 강도를 1~10점으로 정량화해 변화를 추적합니다.

Hybrid 인지 왜곡 분류

질문 기반 대안 사고 제시

왜곡 판단 근거와 재구성 문장을 함께 제공합니다.

Gemini API 설정

Gemini 연결 시 실제 모델 분석으로 전환됩니다

브라우저에서 Gemini REST API를 직접 호출합니다. 키가 없거나 호출에 실패하면 로컬 분석으로 자동 전환됩니다.

연결 설정

기본 모델은 `gemini-2.5-flash`로 설정되어 있습니다.

저장된 API 키 없음
